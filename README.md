# Brasil-E-Commerce-2016-2018-
Análise de dados do e-commerce brasileiro com a base pública da Olist (2016–2018), explorando faturamento, vendas por estado, frete, atrasos, métodos de pagamento e satisfação do cliente. O projeto destaca padrões regionais, impacto logístico e insights de negócio por meio de dashboards e modelagem relacional.
