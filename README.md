# Brasil E-Commerce (2016–2018) | Análise de Dados com Olist

## Visão geral

Este projeto apresenta uma análise exploratória e visual do mercado de e-commerce brasileiro com base no dataset público da **Olist**, disponível no Kaggle. O objetivo foi transformar dados transacionais em informações de negócio, com foco em **faturamento**, **distribuição geográfica das vendas**, **métodos de pagamento**, **prazo de entrega**, **atrasos**, **frete** e **satisfação do cliente**.

A proposta do projeto foi ir além da simples construção de gráficos, buscando responder perguntas relevantes para operação e estratégia, como:

- Como o faturamento evoluiu ao longo do tempo?
- Quais estados concentram o maior volume de vendas?
- Quais métodos de pagamento predominam?
- Existe relação entre atraso na entrega e satisfação do cliente?
- Quais regiões sofrem mais com atrasos?
- Quais categorias apresentam maior peso de frete?
- Como volume de pedidos e satisfação se comportam por categoria?

---

## Dataset utilizado

Base pública da Olist no Kaggle:

**Brazilian E-Commerce Public Dataset by Olist**

O conjunto reúne informações de pedidos realizados entre **2016 e 2018**, contendo dados sobre clientes, pedidos, pagamentos, avaliações, produtos, vendedores e geolocalização.
